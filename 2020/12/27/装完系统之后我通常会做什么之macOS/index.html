下载安装软件

Homebrew

必备！！！

安装之前提前修改仓库地址

Homebrew 源代码仓库中科大镜像
http://mirrors.ustc.edu.cn/help/brew.git.html

安装 Homebrew 会一起安装 Git 和 Python3，Nice ！！！省事！～

The Unarchiver

一款好用的压缩软件，记得关联一下 7z 格式。

通过 App Store 下载

Google Chrome

程序员最常用的浏览器

下载地址：https://www.google.cn/intl/zh-CN/chrome/

Chrome 网上应用店 推荐插件：

  • Vue.js devtools

  • React Developer Tools

  • JSON-handle （一位老师推荐的，用过之后感觉非常不错）

  • Google 翻译

  • Wappalyzer （分析网站使用到到技术）

Visual Studio Code

几乎支持所有项目的开发，特别适合前端开发，小巧的代码编辑器。

插件推荐：

  • Chinese (Simplified) Language Pack for Visual Studio Code

  • vscode-icons

  • GitLens — Git supercharged

  • Python

  • Vetur

  • YAML

  • Docker

  • Prettier - Code formatter

    • Prettier: Arrow Parens
      当箭头函数仅有一个参数时加上括号
      • avoid
    • Prettier: Jsx Single Quote
      • 在 JSX 中使用单引号而不是双引号
    • Prettier: Package Manager
      The package manager you use to install node modules.
      • yarn
    • Prettier: Semi
      • 是否在每行末尾添加分号
    • Prettier: Single Quote
      • 如果为 true，将使用单引号而不是双引号
    • Prettier: Trailing Comma
      尽可能控制尾随逗号的输出。 有效选项： none - 无尾随逗号，es5 - 在 ES5 中有效的尾随逗号（对象，数组等，all - 尾随逗号 尽可能（函数参数）
      • none

下载地址：https://code.visualstudio.com/

JetBrains

JetBrains 家的 IDE 绝对的强大，好用。要装就装全家桶，别和我说什么装一个 IDEA 就够了，不不不，我就喜欢装全家桶。

下载地址：

  • IntelliJ IDEA

  • GoLand

  • PhpStorm

  • PyCharm

  • WebStorm WebStorm 是专门开发前端的 IDE，相比于 VS Code，个人感觉反而复杂了，更加喜欢 VS Code 清新的界面。

Navicat

Navicat Premium 是一套数据库开发工具，让你从单一应用程序中同时连接 MySQL、MariaDB、MongoDB、SQL Server、Oracle、PostgreSQL 和 SQLite 数据库。它与 Amazon RDS、Amazon Aurora、Amazon Redshift、Microsoft Azure、Oracle Cloud、MongoDB Atlas、阿里云、腾讯云和华为云等云数据库兼容。你可以快速轻松地创建、管理和维护数据库。

下载地址：http://www.navicat.com.cn/products/navicat-premium

有一种破解方法：在 GitHub 上搜索 navicat，会出现一些工具。其中 DoubleLabyrinth 大佬的方法是最简单最好用的方法，现在大佬换了一个新账户HyperSine。

仓库： https://github.com/HyperSine/how-does-navicat-encrypt-password

FFmpeg

FFmpeg 是视频处理最常用的开源软件。

相关教程可以阅读 阮一峰 写的一篇关于 FFmpeg 视频处理入门教程 的文章。

下载地址：https://www.ffmpeg.org/download.html

OBS Studio

开源的推流软件，还可以录制视频。

录制视频建议把格式修改为 mp4 格式。

设置 > 输出 > 录像 > 格式 > 修改为 mp4。

下载地址：https://obsproject.com/

Postman

API 工具，模拟发送请求，检查响应并轻松调试。

下载地址：https://www.postman.com/downloads/
